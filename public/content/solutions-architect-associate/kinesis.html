Kinesis Exam Tips

Streams uses shards, each with a set max tps. Firehose does not use shards.

Kinesis Stream Limits

  • Max retention is 7 days, but default is 24 hours. Range is 1-7 days.
  • Max size of data blob is 1MB
  • Consumer - gets data records from Kinesis streams
  • Producer - puts data records into Kinesis streams
  • Streams - ingests a large amount of data in real time, durably stores the data, and makes the data available for consumption
