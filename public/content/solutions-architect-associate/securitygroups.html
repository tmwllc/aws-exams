Security Groups Tips

  • All inbound traffic is blocked by default.
  • All outbound traffic is allowed by default.
  • Changes to security groups take effect immediately.
  • You can have unlimited EC2 instances within a security group.
  • You can have multiple security groups attached to EC2 instances.
  • Security groups are stateful: when creating inbound traffic rules, the same is automatically allowed for outbound.
  • You cannot block specific IPs with Security Groups. You must use Access Control Lists for that.
