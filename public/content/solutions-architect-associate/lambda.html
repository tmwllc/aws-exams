Lambda Tips

Lambda is a compute service that runs your code in response to events and autommatically manages the underlying compute resources for you.

Lambda can automatically run code in response to modifications to objects in S3 buckets, messages arriving in Kinesis streams, or table updates in DynamoDB.

Runs your code on high availability compute infrastructure and performs all the administration of the compute resources, including server and operating system maintenance, capacity provisioning and automatic scaling, code and security patch deployment, and code monitoring and logging. All you need to do is supply the code.

Pricing is based on number of requests as well as duration. Duration is rounded up to nearest 100ms.
