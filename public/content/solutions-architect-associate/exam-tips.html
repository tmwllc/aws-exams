VM Import/Export enables you to import VM images from existing environment to EC2 instances and export them back out.

Services that allow customer to retain full admin priveleges of underlying EC instances

  • Elastic Beanstalk
  • Elastic Map Reduce

If you need OS privileges on an RDS server with high availability, use EC2 instances in a replication configuration utilizing two different AZs.

AZ names are unique per account and do not represent a specific set of resources. us-west-1a for account 1 is not necessarily the same thing as us-west-1a in account 2.
