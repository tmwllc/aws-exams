Elastic Load Balancer Tips

Elastic Load Balancer FAQ

  • Application Load Balancer - newer, introduced in mid 2016, not featured on tests yet
  • Classic Load Balancer
  • Never have IP addresses, only DNS names, which point to a dynamic IP
  • Instances monitored by ELBs will be reported as InService or OutofService
  • Health checks check health by talking to, looking for a specific endpoint and whether the response is 200 (or some other specified value from 200-299)
