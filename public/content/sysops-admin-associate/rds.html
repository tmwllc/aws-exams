Monitoring RDS Exam Tips

  • Metrics
    • BinLogDiskUsage
    • CPUUtilization
    • DatabaseConnections - if app is running slowly, check if DB connections are being closed when done
    • DiskQueueDepth - # of read/write operations queued, always want this to be zero
    • Freeable Memory
    • FreeStorageSpace - Shows if RDS is running out of storage space. Good idea to set an alarm when this gets low.
    • ReplicaLag (seconds) - Lag in seconds between RDS and Read Replicas. If data seems to be out of date, check ReplicaLag to ensure it's low.
    • SwapUsage
    • ReadIOPS
    • WriteIOPS
    • ReadLatency
    • WriteLatency
    • ReadThroughput
    • WriteThroughput
    • NetworkReceiveThroughput
    • NetworkTransmitThroughput
  • Events
    • Events - Let you view a history of events for your DBs, like when backups have started/completed
    • Event Subscriptions - allow you to setup notifications for when events happen. Example, get an SNS notification when DB fails over.
